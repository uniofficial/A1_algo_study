# 🧠 A1 알고리즘 스터디 공지

## 📅 스터디 진행 방식
- 매주 수요일 오후 6시 디스코드 회의방에서 비대면으로 진행

1. 화요일 자정 전까지 문제 풀이 후 규칙 지켜 코드 업로드 
2. 코드 리뷰
3. 매주 수요일 발표

---

## 📁 폴더 구조

weekXX/영문이름/문제코드.java

---

## 🔄 Git 프로세스

1. main 브랜치 최신화
  git pull origin main

2. 개인 브랜치로 이동
  git checkout feat/이름

3. 문제 풀이 후 커밋
  git commit -m "feat: 1234 풀이"

4. 원격 저장소에 push
  git push origin feat/이름

5. Pull Request 생성 -> 코드 리뷰

---

## 📌 규칙

1. commit 메세지: [문제 출처(플랫폼)] / 문제이름
  - 예시) git commit -m "[PG] 정윤희 선인장 숨기기"
2. main에 직접 push 금지
3. PR 제목: n주차 본인이름 문제이름
  - 예시) 1주차 정윤희 선인장 숨기기
4. 문제 풀이 및 PR은 매주 화요일 자정 전까지
5. 코드 리뷰는 수요일 스터디 시간 전까지
